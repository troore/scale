\begin{slide}\heading{Generation Interface}
\begin{itemize}
\item Souce language independent
\item Target architecture independent
\item Extensible
\item Support generation of high-level representations 
\item Prevent user errors where possible
\item Support C, C++, Modula-3, and Fortran~77, and consider
Java, Ada, and parallel Fortran variants
\end{itemize}
\end{slide}

%------------------------------------------------------------------------------
\begin{slide}\heading{Division of labor}
\begin{minipage}[t]{3.5in}
Front End performs:
\begin{itemize}
\item Syntactic and semantic error checking
\item Semantic resolution
\item Enumeration constant assignment
\end{itemize}
\end{minipage}

\begin{minipage}[t]{3.5in}
Back End performs:
\begin{itemize}
\item Memory layout
\item Initialization code generation
\end{itemize}
\end{minipage}
\end{slide}

\begin{slide}\heading{Operators}
As you would expect:
\begin{itemize}
\item Expression negative(Expression e)
\item Expression addition(Expression e1, Expression e2)
\item Expression subtraction(Expression e1, Expression e2)
\item Expression multiplication(Expression e1, Expression e2)
\item Expression division(Expression e1, Expression e2)
\end{itemize}
\end{slide}

%------------------------------------------------------------------------------
\begin{slide}\heading{Statements}
High-level and basically what you would expect:
\begin{itemize}
\item Statement for\_loop(Expression e, Expression e, Expression e, Statement s)
\item Statement for\_loop(Identifier id, Expression e, Expression e, Expression e, Statement s)
\item Statement for\_loop(Declaration d, Expression e, Expression e, Statement s)
\item void block\_begin()
\item Statement block\_end()
\end{itemize}
\end{slide}

%------------------------------------------------------------------------------
\begin{slide}\heading{Declarations}
\begin{itemize}
\item Declarations: Types, Values, Labels, Functions, Exceptions
\item Types: separation of constructors from other attributes
\item Obsessive distinction between description and declaration
\item Values: variables, constants, fields, enumeration elements
\end{itemize}
\end{slide}
