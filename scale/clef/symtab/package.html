Implements a symbol table for use in constructing the Clef AST. A {@link scale.clef.symtab.Symtab Symtab} instance is a container for a tree of {@link scale.clef.symtab.SymtabScope scopes}. Each scope has a set of children scopes and a symbol table to implement the tree and provide scoping of names. A symbol table is a mapping from names to a set of {@link scale.clef.symtab.SymtabEntry declarations}.
